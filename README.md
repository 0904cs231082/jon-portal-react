Job Portal Web Application

A modern Job Portal Web Application built using React.js.

Features

- View Job Listings
- Search Jobs
- Dynamic Routing
- Job Details Page
- Responsive UI
- API Integration
- Reusable Components

---

Tech Stack

- React.js
- React Router DOM
- Axios
- CSS

---

Folder Structure

src/
│
├── components/
├── pages/
├── services/
├── styles/
├── App.jsx
├── main.jsx

---

Installation

Clone the repository:

git clone 

Install dependencies:

npm install

Run the project:

npm run dev
